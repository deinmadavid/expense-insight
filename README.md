# Expense Insight

A React + TypeScript budget tracker for logging income and expenses, monitoring category spend, and exporting monthly activity.

## Features

- Add income and expense entries
- View running income, expense, and balance totals
- Filter the ledger by month
- Inspect category spend with lightweight chart bars
- Export the filtered month to CSV
- Persist entries in local storage

## Run locally

```bash
npm install
npm run dev
```

## Stack

- React 19
- TypeScript
- Vite
